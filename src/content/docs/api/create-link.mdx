# Create Link

Create a new shortened link with custom SEO metadata and optional expiration date.

## Endpoint

```
POST /api/dashboard/link
```

## Authentication

Requires authentication. Include your session cookie or API key in the request.

## Request Body

```json
{
  "url": "https://example.com/very-long-url",
  "expiration": "2024-12-31T23:59:59Z",
  "seo": {
    "title": "Custom Page Title",
    "description": "Custom page description for SEO"
  }
}
```

### Parameters

| Parameter         | Type   | Required | Description                                    |
| ----------------- | ------ | -------- | ---------------------------------------------- |
| `url`             | string | Yes      | The URL to shorten (must be a valid URL)       |
| `expiration`      | string | No       | ISO 8601 datetime when the link should expire  |
| `seo.title`       | string | No       | Custom title for the shortened link page       |
| `seo.description` | string | No       | Custom description for the shortened link page |

## Response

### Success Response (201 Created)

```json
{
  "id": "abc123def",
  "url": "https://example.com/very-long-url",
  "expiration": "2024-12-31T23:59:59.000Z",
  "clicks": 0,
  "createdAt": "2024-01-15T10:30:00.000Z",
  "updatedAt": "2024-01-15T10:30:00.000Z",
  "seo": {
    "id": "seo123",
    "linkId": "abc123def",
    "title": "Custom Page Title",
    "description": "Custom page description for SEO",
    "createdAt": "2024-01-15T10:30:00.000Z",
    "updatedAt": "2024-01-15T10:30:00.000Z"
  }
}
```

### Error Responses

#### 400 Bad Request

```json
{
  "error": {
    "message": "Invalid URL format"
  }
}
```

#### 401 Unauthorized

```json
{
  "error": {
    "message": "Authentication required"
  }
}
```

## Example Usage

```javascript
const response = await fetch("/api/dashboard/link", {
  method: "POST",
  headers: {
    "Content-Type": "application/json",
    Authorization: "Bearer YOUR_API_KEY",
  },
  body: JSON.stringify({
    url: "https://example.com/very-long-url",
    expiration: "2024-12-31T23:59:59Z",
    seo: {
      title: "My Custom Link",
      description: "This is a custom shortened link",
    },
  }),
});

const link = await response.json();
console.log("Created link:", link);
```

## Notes

- The generated link ID is 9 characters long and URL-safe
- If no expiration date is provided, the link will never expire
- SEO metadata is optional but recommended for better link presentation
- The link will be immediately available for use after creation
